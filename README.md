# Product Failure Analysis – Machine Failure Dataset

## Course:
Critical Thinking, Design Thinking, Leadership and Teamwork

## Task:
Task #01 – Product Failure Analysis

## Objective:
Analyze machine failure conditions using dataset features.

## Dataset Features:
- Air Temperature
- Process Temperature
- Rotational Speed
- Torque
- Tool Wear

## Key Observations:
- High torque (>70 Nm) leads to failures
- Tool wear above 200 min causes breakdown
- High process temperature causes overheating

## Conclusion:
Failures are mainly caused by:
1. Excessive tool wear
2. High torque load
3. Overheating

## Fix Recommendations:
- Regular tool replacement
- Load balancing
- Cooling system maintenance
